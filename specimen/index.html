    MENGUKUR JARAK DATA

    Mengukur Jarak Data¶

    A. Mengukur Jarak Tipe Numerik¶

    1. Minkowski Distance¶

    ​ Minkowski Distance merupakan metrik pada ruang vektor normed yang bisa dianggap sebagai generalisasi dari jarak euclidean dan jarak manhattan. $$ d _ { \operatorname { min } } = ( sum _ { i = 1 } ^ { n } | x _ { i } - y _ { i } | ^ { m } ) ^ { \frac { 1 } { m } } , m \geq 1 $$ Keterangan :

    m = bilangan riel positif

    xi dan yi = dua vektor dalam runang dimensi n

    n = banyak atribut

    2. Manhattan Distance¶

    ​ Manhattan distance adalah kasus dari jarak Minkowski distance untuk m = 1. Manhattan distance juga sensitif pada outlier. $$ d _ { \operatorname { man } } = \sum _ { i = 1 } ^ { n } \left| x _ { i } - y _ { i } \right| $$ Keterangan :

    xi dan yi = dua vektor dalam runang dimensi n

    n = banyak atribut

    3. Euclidean Distance¶

    ​ Euclidean distance adalah perhitungan jarak dari 2 buah titik dalam Euclidean Space. Jarak Euclidean berkinerja baik ketika digunakan untuk kumpulan data cluster kompak atau terisolasi.

    Kelemahan euclidean distance :

    a. Jika dua vektor data tidak memiliki nilai atribut yang sama, kemungkin memiliki jarak yang lebih kecil daripada pasangan vektor data lainnya yang mengandung nilai atribut yang sama.

    b. Fitur skala terbesar akan mendominasi yang lain. Sehingga diperlukan normalisasi fitur kontinu untuk mengatasi kelemahan tersebut.

    4. Average Distance¶

    ​ Average distance adalah modikfikasi dari jarak Euclidian untuk memperbaiki hasilnya. Average distance juga dapat diartikan hasil akar dari 1 dibagi banyak data yang dikali dengan xi-yi kuadrat. $$ d _ { a v e } = \left ( \frac { 1 } { n } \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } \right) ^ { \frac { 1 } { 2 } } $$ Keterangan :

    xi dan yi = dua vektor dalam runang dimensi n

    n = banyak atribut

    5. Weighted Euclidean Distance¶
    d _ { w e } = \left ( \sum _ { i = 1 } ^ { n } w _ { i } ( x _ { i } - y _ { i } \right) ^ { 2 } ) ^ { \frac { 1 } { 2 } }

    Keterangan :

    wi = bobot yang diberikan pada atribut ke i.

    6. Chord Distance¶

    ​ Chord distance yaitu ukuran jarak modifikasi dari Euclidean distance. Chord distance dimodifikasi untuk mengatasi kekurangan dari Euclidean distance. Chord distance dapat dipecahkan dengan menggunakan skala pengukuran yang baik. Jarak ini dapat juga dihitung dari data yang tidak dinormalisasi. Berikut adalah rumus chord distance. $$ d _ { \text {chord} } = \left ( 2 - 2 \frac { \sum _ { i = 1 } ^ { n } x _ { i } y _ { i } } { | x | _ { 2 } | y | _ { 2 } } \right) ^ { \frac { 1 } { 2 } } $$ dimana ∥x∥2‖x‖2 adalah $$ L^{2} \text {-norm} | x |{2} = \sqrt { \sum{ i = 1 }^{ n }x_{i}^{2}} $$

    7. Mahalanobis Distance¶

    ​ Jarak Mahalanobis yang teratur dapat digunakan untuk mengekstraksi hyperellipsoidal clusters. Jarak Mahalanobis dapat mengurangi distorsi yang disebabkan oleh korelasi linier antara fitur dengan menerapkan transformasi pemutihan ke data atau dengan menggunakan kuadrat Jarak mahalanobis. Mahalanobis distance yaitu akar dari x-y dikali dengan S invers dan dikali dengan x-y transpose atau dapat dirumuskan sebegai berikut. $$ d _ { m a h } = \sqrt { ( x - y ) S ^ { - 1 } ( x - y ) ^ { T } } $$ dengan S yaitu matrik covariance data.

    8. Cosine Measure¶

    ​ Cosine Similarity dapat diimplementasikan untuk menghitung nilai kemiripan antar kalimat dan menjadi salah satu teknik untuk mengukur kemiripan teks yang popular. Berikut adalah rumus untuk mencari cosine measure. $$ Cosine(x,y)=\frac { \sum _ { i = 1 } ^ { n } x _ { i } y _ { i } } { | x | _ { 2 } | y | _ { 2 } } $$ dimana ∥y∥2‖y‖2 adalah Euclidean norm dari vektor y=(y1,y2,…,yn)y=(y1,y2,…,yn) didefinisikan dengan $$ |y|_{2}=\sqrt{ y _ { 1 } ^ { 2 } + y _ { 2 } ^ { 2 } + \ldots + y _ { n } ^ { 2 } } $$

    9. Pearson Correlation¶

    ​ Koefisien korelasi yaitu mengukur kekuatan dan arah hubungan linier dari dua veriabel. Perlu diingat bahwa nilai koefisien korelasi yang kecil (tidak signifikan) bukan berarti kedua variabel tersebut tidak saling berhubungan. Mungkin saja dua variabel mempunyai keeratan hubungan yang kuat namun nilai koefisien korelasinya mendekati nol, misalnya pada kasus hubungan non linier. Dengan demikian, koefisien korelasi hanya mengukur kekuatan hubungan linier dan tidak pada hubungan non linier. Kelemahan pearson correlation adalah sensitif terhadap outlier $$ Pearson ( x , y ) = \frac { \sum _ { i = 1 } ^ { n } ( x _ { i } - \mu _ { x } ) ( y _ { i } - \mu _ { y } ) } { \sqrt { \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } } \sqrt { \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } } } $$

    B. Mengukur Jarak Atribut Binary¶

    Atribut biner hanya memiliki dua status: 0 dan 1 , berarti 0 tidak ada dan 1 berarti ada. Binary attribute biasanya diartikan sebagai Boolean jika kedua keadaannya adalah true(benar) dan false(salah). Binary attribute bisa simetris(symmetric) dan bisa asimetris(assymmetric). Simetris jika kedua nilainya bernilai sama/setimbang harganya, sehingga tidak bisa diberi kode 0 atau 1, sedangkan asimetris kedua nilainya tidak setimbang harganya, sehingga dapat diberi kode 0 atau 1. Contoh :

    • Status kehidupan : hidup atau meninggal

    • Hasil tes ujian : lulus, tidak lulus

    • Tes malaria : positif, negatif

    Jika objek i dan j dinyatakan sebagai atribut biner simetris, maka dissimilarity antar i dan j adalah

    d ( i , j ) = \frac { r + s } { q + r + s + t }

    Keterangan :

    q = jumlah atribut yang sama dengan 1 untuk kedua objek i dan j

    r = jumlah atribut yang sama dengan 1 untuk objek i tetapi 0 untuk objek j

    s = jumlah atribut yang sama dengan 0 untuk objek i tetapi 1 untuk objek j

    t = jumlah atribut yang sama dengan 0 untuk kedua objek i dan j.

    ​ Untuk atribut biner asimetris, kedua kondisi tersebut tidak sama pentingnya. Diberikan dua atribut biner asimetris, pencocokan keduanya 1 (kecocokan positif) kemudian dianggap lebih signifikan daripada kecocokan negatif. Ketidaksamaan berdasarkan atribut-atribut ini disebut asimetris biner dissimilarity, di mana jumlah kecocokan negatif, t, dianggap tidak penting dan dengan demikian diabaikan. Berikut rumus untuk atribut biner asimetris. $$ d ( i , j ) = \frac { r + s } { q + r + s } $$ Selain itu kita juga dapat menghitung perbedaan antara dua atribut biner berdasarkan pada disimilarity. $$ \operatorname { sim } ( i , j ) = \frac { q } { q + r + s } = 1 - d ( i , j ) $$

    C. Mengukur Jarak Tipe Categorical¶

    1. Overlay Metric¶

    ​ Overlay metric yaitu metode yang digunakan untuk menghitung jarak jika semua atributnya bertipe nominal. Berikut adalah rumus untuk menghitung overlay metric. $$ d ( x , y ) = \sum _ { i = 1 } ^ { n } \delta ( a _ { i } ( x ) , a _ { i } ( y ) ) $$ Keterangan :

    n = banyaknya atribut

    ai(x) dan ai(y) = nilai atribut ke i yaitu Ai dari masing masing objek x dan y

    δ (ai(x),ai(y)) = 0 jika ai(x) = ai(y) dan 1 jika sebaliknya.

    2. Value Difference Metric (VDM)¶

    ​ Value difference metric adalah salah satu metode untuk mengukur jarak tipe categorical. Berikut merupakan rumus value diffrence metric. $$ d ( x , y ) = \sum _ { i = 1 } ^ { n } \sum _ { c = 1 } ^ { C } \left| P ( c | a _ { i } ( x ) ) - P ( c | a _ { i } ( y ) ) \right | $$ Keterangan :

    C = banyaknya kelas

    P(c|ai(x)) = probabilitas bersyarat dimana kelas x adalah c dari atribut Ai, yang memiliki nilai ai(x)

    P(c|ai(y)) = probabilitas bersyarat dimana kelas y adalah c dengan atributAi memiliki nilai ai(y)

    ​ VDM mengasumsikan bahwa dua nilai dari atribut lebih dekat jika memiliki klasifikasi sama. Pendekatan lain berbasis probabilitas adalah SFM (Short and Fukunaga Metric). $$ d ( x , y ) = \sum _ { c = 1 } ^ { C } \left | P ( c | x ) - P ( c | y ) \right| $$ dengan syarat probabilitas keanggotaan kelas diestimasi dengan P(c|x) dan P(c|y) didekati dengan Naive Bayes.

    3. Minimum Risk Metric (MRM)¶

    ​ MRM berbeda dengan SFM yang meminimumkan selisih antara kesalahan berhingga dan kesalahan asymtotic. MRM yaitu meminimumkan risk of misclassification. Berikut adalah rumus MRM. $$ d ( x , y ) = \sum _ { c = 1 } ^ { C } P ( c | x ) ( 1 - P ( c | y ) ) $$

    D. Mengukur Jarak Tipe Ordinal¶

    ​ Ordinal attribute adalah atribut dengan nilai-nilai yang memiliki urutan yang mempunyai arti atau tingkatan, akan tetapi jarak antara nilai-nilainya tidak diketahui. Ordinal attribute berguna untuk mendaftarkan taksiran suatu kualitas yang tidak bisa diukur secara objektif. Oleh karena itu ordinal attribute biasanya digunakan dalam survey atau rating.

    Contoh :

    • Nilai ujian (A, B, C, D, E)
    • Semester pada perkuliahan (I, II, III, dst)
    • Kualitas air (sangat baik, baik, cukup, buruk, sangat buruk)
    • Ukuran baju (small, medium, large, extra large)

    ​ Perlakuan untuk atribut ordinal sama dengan atribut numerik pada saat menghitung disimilarity antara objek. Misalkan f adalah atribut-atribut dari atribut ordinal dari n objek. Menghitung disimilarity terhadap f fitur sebagai berikut:

    • Nilai f untuk objek ke-i adalah xif, dan f memiliki Mf status urutan , mewakili peringkat 1,..,Mf1,..,Mf Ganti setiap xif dengan peringkatnya, rif∈{1...Mf}.

    • Karena setiap atribut ordinal dapat memiliki jumlah state yang berbeda, diperlukan untuk memetakan rentang setiap atribut ke [0,0, 1.0] sehingga setiap atribut memiliki bobot yang sama. Perlu melakukan normalisasi data dengan mengganti peringkat rifrif dengan $$ z _ { i f } = \frac { r _ { i f } - 1 } { M _ { f } - 1 } $$

    • Dissimilarity kemudian dihitung dengan menggunakan ukuran jarak seperti atribut numerik dengan data yang baru setelah ditransformasi z if.

    E. Menghitung Jarak Tipe Campuran¶

    ​ Salah satu teknik tersebut menggabungkan atribut yang berbeda ke dalam matriks ketidaksamaan tunggal dan menyatakannya dengan skala interval antar [0,0,1.0]. Misalkan data berisi atribut p tipe campuran. Ketidaksamaan (disimilarity ) antara objek i dan j dapat dinyatakan dengan rumus berikut. $$ d ( i , j ) = \frac { \sum _ { f = 1 } ^ { p } \delta _ { i j } ^ { ( f ) } d _ { i j } ^ { ( f ) } } { \sum _ { f = 1 } ^ { p } \delta _ { i j } ^ { ( f ) } } $$ dengan catatan δijf=0 - jika xif atau xjf adalah hilang (i.e., tidak ada pengukuran dari atribut f untuk objek i atau objek j)

    #jarak binary
def binaryDist(v1,v2,jnis):
    q=0
    r=0
    s=0
    t=0
    for x in range (len(jnis)):
        if (int(k.values.tolist()[v1][jnis[x]]))==1 and (int(k.values.tolist()[v2][jnis[x]]))==1:
            q=q+1
        elif (int(k.values.tolist()[v1][jnis[x]]))==1 and (int(k.values.tolist()[v2][jnis[x]]))==2:
            r=r+1
        elif (int(k.values.tolist()[v1][jnis[x]]))==2 and (int(k.values.tolist()[v2][jnis[x]]))==1:
            s=s+1
        else:
            t=t+1
    return ((r+s)/(q+r+s+t))
