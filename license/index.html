    LINEAR REGRESSION

    Regresi Linear¶

    1. Regresi Linear Sederhana¶

    Regresi linear sederhana merupakan pendekatan untuk memodelkan hubungan antara variable terikat Y dan satu atau lebih variable bebas yang disebut X.

    Berikut merupakan persamaan umum dari regresi linear sederhana. $$ Y= a +bx $$ Di bawah ini merupakan rumus untuk mencari nilai a dan b : $$ a = \frac{\sum y(\sum x^{2})-\sum x\sum y}{n\sum x^{2}-(\sum x)^{2}} $$

    b= \frac{n\sum xy - \sum x \sum xy}{n\sum x^{2} - (\sum x)^{2}}

    Keterangan :

    Y = variabel terikat

    X = variabel bebas

    a = konstanta

    b = nilai koefisien regresi

    2. Persamaan Linear Berganda¶

    Regresi linear sederhana merupakan pendekatan untuk memodelkan hubungan antara variable terikat Y dan satu atau lebih variable bebas yang disebut X. Biasanya pada regresi linear berganda ini tidak hanya memiliki satu variebel bebas tetapi bisa lebih dari satu variabel.

    Berikut merupakan persamaan umum regresi linear berganda. $$ Y = a+b_{1}x_{1}+b_{2}x_{2}+b_{3}x_{3}.....+b_{n}x_{n} $$ Di bawah ini merupakan rumus untuk mencari nilai a, b1, dan b2 : $$ a= \frac {\sum y}{n}-b_{1}(\frac {\sum x_{1}}{n})-b_{2}(\frac {\sum x_{2}}{n}) $$

    b1 = \frac{(\sum x_{2}^{2})(\sum x_{1}y)-(\sum x_{1}x_{2})(\sum x_{2}y)}{(\sum x_{1}^{2})(\sum x_{2}^{2})-(\sum x_{1}x_{2})^{2}}
    b2 = \frac{(\sum x_{1}^{2})(\sum x_{1}y)-(\sum x_{1}x_{2})(\sum x_{1}y)}{(\sum x_{1}^{2})(\sum x_{2}^{2})-(\sum x_{1}x_{2})^{2}}

    Keterangan :

    Y = variabel terikat

    a = konstanta

    X (1,2,3......) = variabel bebas

    b (1,2,3......) = nilai koefisien regresi
